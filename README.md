# Inventory-Management-Dashboard-Excel-Project
This project showcases a fully interactive Excel Inventory Dashboard built using PivotTables, PivotCharts, and Slicers. The goal was to analyze inventory levels, monitor stock performance, identify low-stock risks, and understand product movement across suppliers and locations.


---

## Tools Used

Microsoft Excel (Pivot Tables, Pivot Charts, Slicers)

Excel Table for dynamic data modeling

Conditional formatting

KPI Cards & Dashboard layout design



---

## Project Objectives

Track total inventory value and total units in stock

Identify low-stock items

Analyze supplier contribution to inventory cost

Understand product lead time performance

Break down stock quantity by location and product type

Build a fully interactive dashboard using slicers



---

## Key Insights

66 items are currently in low-stock status

Total inventory contains 125,841 units

Total inventory value stands at $12.8M+

Certain suppliers contribute disproportionately to inventory cost

Product lead time varies significantly across categories

Location breakdown shows uneven inventory distribution



---

## Dashboard Features

Interactive Slicers (Product Category, Supplier, Storage Location, Lead Time Status)

KPI Cards for quick, high-level analysis

Donut Charts for category and supplier breakdown

Bar Charts for product value and lead time

Low Stock Alerts for quick decision-making

All charts and KPIs automatically update with slicer selections
